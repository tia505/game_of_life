# Conway's Game of Life
https://playgameoflife.com/

# Explanation
Cells live and die by the amount of neighbors they have in the 8 matrix spaces around them.

# Rules
No. Neighbors   | Cell Status
----------------|-----------
0, 1, 4+        | Dead
2, 3            | Alive
3               | Dead -> Alive
